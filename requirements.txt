# Python 3.10 trở lên — bắt buộc. Tool sử dụng cú pháp PEP 604:
#   - `list[str] | None`
#   - `dict[str, tuple[...]]`
#   - match/case ở một số nơi
#
# Kiểm tra:
#   python --version
#
# -----------------------------------------------------------------------------
# Công cụ ngoài (cài riêng, không qua pip)
# -----------------------------------------------------------------------------
#
# 1. Trivy  — BẮT BUỘC. Dùng để scan SBOM (vuln + license) và fallback
#             `trivy fs --format cyclonedx` khi generator riêng fail.
#             Tải tại: https://trivy.dev/latest/getting-started/installation/
#
# 2. Maven  — khuyến nghị cho dự án Maven. Ưu tiên `mvnw` / `mvnw.cmd` trong
#             project, fallback `mvn` trong PATH.
#
# 3. Gradle — khuyến nghị cho dự án Gradle. Ưu tiên `gradlew` trong project,
#             fallback `gradle` trong PATH.
#
# 4. Node.js + npm + npx — bắt buộc cho dự án Node.js
#             (dùng `@cyclonedx/cyclonedx-npm`).
#
# 5. .NET SDK + dotnet-CycloneDX — bắt buộc cho dự án .NET. Tool sẽ tự
#             `dotnet tool install --global CycloneDX` nếu thiếu.
#
# 6. Flutter — khuyến nghị cho dự án Flutter (dùng `pubspec.yaml`).
#
